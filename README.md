# Vending Machine React App

### Tech Stack

- React js
- json-server
- TailwindCSS

### json-server

##### install

- npm install -D json-server

##### Run

- npm run server
- json-server --watch db.json --port 3001

### Route

- http://localhost:3001/products

### Sample json

- {id: '1', title: 'Donat', image: 'donut.png', stock: 5, price: 10000}
